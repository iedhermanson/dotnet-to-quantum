# Week 4 – Multi-Qubit Systems

## Goal

Work with tensor products to represent multi-qubit states.

## Concepts

- Tensor (Kronecker) product: ⊗
- Building multi-qubit states from single qubits
- Computational basis for n qubits: |00⟩, |01⟩, |10⟩, |11⟩, ...
- Multi-qubit gates:
  - **CNOT**: Controlled-NOT (creates entanglement)
  - **Identity**: I (no operation)
- Tensor products of gates: H ⊗ I, etc.
- Entanglement and Bell states

## Practice in .NET

1. Implement a tensor product (Kronecker) function
2. Simulate a 2-qubit system (4-dimensional state space)
3. Apply CNOT gate to create entanglement
4. Create a Bell state: (|00⟩ + |11⟩)/√2
