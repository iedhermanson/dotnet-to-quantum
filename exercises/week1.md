# Week 1 – Foundations & Mindset

## Goal

Build intuition for vectors, matrices, and complex numbers.

## Concepts

- Scalars, vectors, matrices
- Complex numbers
- Basic operations (addition, scalar multiplication, dot products)

## Practice in .NET

1. Install Math.NET Numerics
2. Create and manipulate vectors/matrices
3. Implement vector addition, scalar multiplication, and dot products
