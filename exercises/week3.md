# Week 3 – Quantum Gates as Matrices

## Goal

Learn how quantum operations are matrix transformations.

## Concepts

- Matrix multiplication and composition
- Unitary matrices (definition & properties)
- Unitary condition: U†U = I (preserves probability)
- Common quantum gates:
  - **Pauli-X**: Bit flip (NOT gate)
  - **Pauli-Y**: Bit flip with phase
  - **Pauli-Z**: Phase flip
  - **Hadamard**: Creates superposition

## Practice in .NET

1. Implement gates as matrices
2. Apply gates to qubit vectors
3. Verify unitarity by checking U†U = I
4. Compose multiple gates
